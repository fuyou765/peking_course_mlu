## AI Course Demo

### 环境准备
请先进入寒武纪Pytorch Docker,以下命令在Docker内部执行:
```
1.激活虚拟环境
source /torch/venv3/pytorch/bin/activate
2.安装python必要模块
pip install -r requirements.txt
```
